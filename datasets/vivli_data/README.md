# Vivli Data

This folder contains documentation and metadata related to Vivli AMR surveillance datasets used for ResistanceRadar research and validation.
Link to download the dataset is :https://drive.google.com/drive/folders/1qKKiKilo4EN4IonfP2yYFnrTv26oTJPF?usp=sharing

## Purpose

These datasets are being evaluated to:

- Study antimicrobial resistance (AMR) trends
- Explore genotype and phenotype relationships
- Compare clinical AMR surveillance data with wastewater-derived AMR signals
- Support statistical modeling and forecasting workflows
