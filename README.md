# ResistanceRadar

ResistanceRadar is a wastewater-based antimicrobial resistance (AMR) surveillance platform.

## Objective

Detect antimicrobial resistance genes (ARGs) from wastewater metagenomic sequencing data and forecast future AMR trends before traditional clinical surveillance.

## Pipeline

FASTQ
↓
Quality Control (fastp)
↓
Read Processing
↓
AMR Detection
↓
Statistical Modeling
↓
Forecasting
↓
Dashboard
