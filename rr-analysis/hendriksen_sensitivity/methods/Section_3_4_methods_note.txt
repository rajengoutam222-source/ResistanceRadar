Stage 3.4 Hendriksen CARD/RGI ARG detection sensitivity analysis

This preliminary research analysis evaluated ARG detection sensitivity for one Hendriksen global wastewater metagenomics sample using CARD/RGI. The analyzed sample was ENA accession ERR1713332 from ERP015409, mapped to Hendriksen sample AUS.18 based on country, city, and collection date metadata.

Input reads were the Stage 3.3 Bowtie2-cleaned paired-end FASTQ files generated after host/contamination removal using the production decontamination reference GRCh38.p14, PhiX174, pUC19, pBR322, and UniVec. The selected cleaned sample contained 13,881,133 retained read pairs.

ARG detection was performed using RGI BWT with KMA against CARD database version 4.0.1. RGI detected 467 gene-level CARD rows and 31 unique CARD/RGI drug classes.

The Hendriksen reference truth set was derived from 41467_2019_8853_MOESM7_ESM.xlsx, sheet ResFind.Gene.count. A Hendriksen ResFinder gene was considered present in AUS.18 if its count was greater than zero. This produced 150 reference ResFinder genes. These genes were mapped to ARG drug classes using predefined ResFinder gene-family rules. The primary sensitivity analysis was performed at the ARG drug-class level, rather than exact gene level, because class-level comparison is more defensible across ResFinder and CARD/RGI nomenclatures.

CARD/RGI beta-lactam subclasses including carbapenem, cephalosporin, monobactam, and penicillin beta-lactam were normalized to beta-lactam antibiotic for comparison with the Hendriksen reference class set.

Sensitivity was defined as:

Sensitivity = reference ARG drug classes recovered by CARD/RGI / total Hendriksen reference ARG drug classes present.

For ERR1713332 / AUS.18, CARD/RGI recovered 11 of 11 Hendriksen reference ARG drug classes, corresponding to 100.00% class-level sensitivity.

This result should be framed as a preliminary one-sample research analysis supporting the feasibility of high-sensitivity ARG recovery from Hendriksen wastewater metagenomes. It should not be described as a validated full ResistanceRadar prototype benchmark.
